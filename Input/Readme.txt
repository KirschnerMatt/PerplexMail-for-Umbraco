This is the amazing PerplexMail package. It will make your email life a lot brighter! 

All information about the package (documentation, package-files, PerplexMail-UmbracoForms-dll, etc.) can be found on our.

After you install you have to do two things:

1) Go to the Content-section in Umbraco and specify your sender-adress on the node "Emails", tab "Default settings".

2) Go to the "Thank you, Perplex!"-mail and select the correct email template.

Now you can start sending mail!
